Geração de Propostas AUTOMATIZADA

Projeto desenvolvido por mim.

Usando HTML, CSS e JavaScript como FRONTEND

Hospedagem em GOOGLE DRIVE E GOOGLE PLANILHAS

Descrição do Projeto: Este sistema foi projetado para automatizar a criação e a gestão de propostas. O foco principal é o uso de uma planilha do Google, onde a coleta de informações de propostas e funcionários pode ser feita de maneira fácil e organizada. As operações CRUD são implementadas para permitir a criação, leitura, atualização e exclusão de registros. Além disso, há um filtro de obras com base em parâmetros específicos, como ID, Nome do Funcionário, Empresa e a obra onde o funcionário está alocado.

Detalhes Técnicos: O projeto segue uma estrutura com códigos que gerenciam TP Números e TP Nomes, sendo o TP Número composto por sequências alfanuméricas, enquanto o TP Nome é formado por letras com acentos especiais.
